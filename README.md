##  Project Overview

This project is a **modern, responsive chat user interface** inspired by popular AI chat applications like ChatGPT and Claude.

The goal of this assignment is to design a **clean, interactive, and mobile-responsive chat interface** using front-end technologies only. No backend or real AI integration is used. Instead, mock AI responses simulate chat functionality.

The project focuses on:

- Modern UI design
- Smooth animations
- Responsive layouts
- Interactive user experience
- Clean and structured code

##  Features Implemented

### 1️ Welcome Screen
- Centered welcome interface
- AI icon / logo
- 4 interactive suggestion cards
- Clean modern layout

### 2️ Chat Interface
- User message bubbles (right aligned)
- AI message bubbles (left aligned)
- Message timestamps
- Smooth fade-in animation

### 3️ Input Area
- Auto-resizing textarea
- Send button with enabled/disabled states
- Attach button with icon
- Footer disclaimer text

### 4️ AI Simulation
- Mock AI responses
- Typing indicator animation
- Delay simulation for realistic response
- Auto-scroll when new message appears

### 5️ Sidebar
- Chat history list
- New chat button
- User profile section
- Fixed width layout

### 6️ Responsive Design
- Mobile friendly layout
- Sidebar slide-in menu
- Adaptive message bubble width
- Works on desktop, tablet, and mobile
